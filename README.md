# C++ 2D Game Engine
The goal of this project is to act as more or less a learning tool. I wrote a fairly basic 2D engine in Java and this is the attempt to port it over to C++. Main reason for this is the high computational cost of running the code in a VM, unparallelized, on a single thread, limiting the performance of the game engine

This project will use CUDA core processing to handle heavy-duty tasks such as collision detection and geometry rotation., and perhaps have CUDA render the game.
## Scope
As of now, the plans of this project are pretty limited in scope. Basically, create a framework by which props and prop components can be extended upon with Java. Low-level data types, abstracted efficiently for access by Java and other higher-level languages

That's it for now!
